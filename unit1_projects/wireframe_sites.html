SoundCloud

What problem is the web page solving?

SoundCloud is a platform for sharing and discovering the world's sounds.

What visual challenges are there?

Visually, the site is very content focused. This creates the visual challenge of
vast information presentation; how will the site be organized in such a way that
content does not look and feel cluttered? Is it obvious to the user which actions
(like, repost, comment) are possible and encouraged through design?

Twitter

What problem is the web page solving?

Twitter is a channel for text and media, allowing users to broadcast to any audience.
It solves the problem of presenting these broadcasts in near-real time within a structured,
readable stream focused on user content.

What visual challenges are there?

Visual challenges exist because some users perform specific actions outside
of simply reading their Twitter stream. Placement of the buttons, text fields, etc.
should be obvious, yet unobtrusive creating a visual challenge for the site's layout.

Quora

What problem is the web page solving?

Quora is a site dedicated to asking and answering questions about any imaginable topic.
It solves the problem of quick and efficient community-based learning, allowing anyone to ask or answer a question about anything.

What visual challenges are there?

The visual challenge with this site is being able to present enough of each answer to garner
the interest and attention of users browsing the site, while still visually presenting enough
content (questions and answers) to demonstrate the utility of the site itself.
